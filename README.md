# HPCL Scholar 🎓

A premium, browser-based exam preparation platform designed specifically for HPCL CBT (Computer Based Test) aspirants. 

## 🚀 Getting Started

No installation required. This is a static web application.

1. Download or clone this repository.
2. Open **`index.html`** in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Start learning from the **Guide** or jump straight into a **Mock Exam**.

## ☁️ Deployment (VPS)

If you want to host this on your own VPS using Docker:

1. ZIP this folder and upload it to your server.
2. Unzip and run: `./setup.sh`
3. Access the app at `http://your-vps-ip:8080`

Ensure you have `docker` and `docker-compose` installed on your VPS.

## ✨ Key Features

- **CBT Simulator**: Realistic exam interface with timer, question navigation, and "Scribble Pad" for calculations.
- **Advanced Analytics**: Track your "Readiness Score", topic-wise accuracy, and syllabus coverage.
- **Psychological Radar**: Automatic classification of mistakes (e.g., "Speed Pressure", "Concept Confusion") to help you understand *why* you are losing marks.
- **Mastery Vault**: Automatically archives topics once you achieve a 3-question correct streak.
- **Revision Guide**: Curated formulas and shortcuts for Quantitative Aptitude, Reasoning, and English.
- **Theme Support**: Includes a "Vintage Library" mode for a focused, paper-like reading experience.

## 📊 Privacy & Data

All your progress, history, and analytics are stored **locally in your browser** (LocalStorage). Your data never leaves your machine. 

To export your results or clear your history, use the actions in the **Analytics** dashboard.
